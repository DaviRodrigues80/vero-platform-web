# VERO Platform Web

## Setup

1. Instale dependências: `npm install`
2. Inicie ambiente de desenvolvimento: `npm run dev`

## Estrutura

- `app/platform` — Console da Plataforma (CEO)
- `app/tenant` — Console do Tenant (Admin)
- `app/auth` — Autenticação
- `app/shared` — Código compartilhado
- `components` — Componentes React
- `services` — Camada de serviços
- `hooks` — React hooks
- `styles` — Estilos
- `public` — Arquivos públicos
- `tests` — Testes

## Variáveis de Ambiente

Veja `.env.example` para referência.
